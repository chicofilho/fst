Understand our calculation

V1.1

The first calculation version was used in the Gold Coast Men's Event. At the women's event the picks would have made 309.24 points

It was our first version and we used several parameters to help with it

  • Actualizated position Rank
  • Participation in previous events
  • Momentum

V1.2

The second version of the Fantasy Bot was built and we hoped to do better on Bells Beach

We still use the same parameters, but we have improved the previous events participations calculation. We basically downgraded surfers that were in a descendent pattern. We will post this rank for players to be aware of it.

On Gold Coast the Fantasy Bot V1.2 has downgraded surfers for the competition.

We use the same data we had previously on each event to calculate the points.

V1.3

The third version is a brand new way of looking at Momentum Rank

For the third version we have improved the calculations for the Momentum Rank, the rookies now start the season with medium scores for this rank and they loose it quickly by scoring bad. For the long run, there was also an improvement for the equation, our statistical analysis for the first two events detected a high importance on Momentum Rank Variables. This will be a major change for the long run. We are still analysing previous data and we have a clue that Momentum Rank has a low influence in Reef Waves, for this reason v1.3 is still unstable for Margaret River.

We are commited to play serious on version comparisons (v1.2 can ponctuate better than v1.3 in Margaret River). If you wanna choose exactly the v1.2 team, just choose overall at Fantasy Simulator and avoid RD1 matchups, this will generate exactly the v1.2 picks.

V1.4

The fourth version has a very important improvement. We have made some equalizations and weight balance for the ranks. This is the first mutable version we have made, it will always update ranks calculations for better as we have more historical clues.

V1.5

Small tune update for version 1.4 with more detailed weight balance for better calculation

V1.6

We tried to use statistical projection and base the picks only in those values. The results were very far from what we expected. We have discontinued this version

V1.7

We isolated the performance variables to point the ones that are more important. No variables regarding Momentum or Past Position were used

V1.8

Small changes, all ranks are used and the performance is tuned based on v1.7 results. We hoped to see an improvement based on those steps, the results were solid, but far from our expectation.
