There was a huge step made in 2016 when we created eight calculation versions. If you with to understand them better, check the first calculation updates page

We have choosed the best two version from last year. A new version is coming, meanwhile you can keep track of how the best version are performing.

V1.7

The version is based only in performance variables such as AHS on the year, venue and career. We take also in consideration the winning percentage and the average points a surfer does on Fantasy games.

V1.5

Is an overall calculation. The calculations used in v1.7 are one of 3 variables used by this calculation (the results from 1.7 are used as an input for this version). In advance it uses performance variables together with momentum and past posisition.
